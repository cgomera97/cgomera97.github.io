Flower Girl
A Handcrafted Letterpress Chapbook
"Flower Girl" is a short story that I wrote which follows a fairytale narrative set in Mexico. For this project, I used it as copy for a letterpress and bookmaking project. The result is a marigold children's book with a condensed version of a story on dealing with grief.

The book features hand-set type from the Globe Press, linocut illustrations, and hand-sewn binding.

It was printed in an edition of 15.
