All Access
A website concept for an Accessible Design event
For this web project, I wanted to challenge myself and the way I think about design through the consideration of those with disabilities.

"All Access" is an event with a panel of speakers where designers can discuss how we can develop work that is accessible to all.

The idea was to create a site that was in itself accessible. So the buttons on the side allow the user to customize according to their disability.
